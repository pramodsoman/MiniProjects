Project-1

Movie Discovery App

This is a simple movie discovery web app that allows users to browse and search for movies to see some basic information and overviews.It also supports browsing popular and searching movies using the TMDB API, allowing users to view basic information and toggle overview details. The code uses asynchronous JS to fetch and render API data. The app has a minimal, easy to use interface with some nice styling effects.

Features:

  • It uses the TMDB (The Movie Database) API to fetch popular movies and display them on the homepage.
  • The getMovies() function fetches the data from the TMDB API URL and passes it to showMovies() to display each movie.
  • For each movie, it displays the title, poster image, vote average, and overview. The CSS stylizes each movie container.
  • There is a search bar that allows the user to search for movies by name. The search calls the TMDB search API and displays matching movies.
  • The getClassByRate() function assigns a color class to the vote average span based on the rating value.Green for 8+, orange for 5+, red for below 5.
  • When a movie poster is clicked, the overview text slides up to reveal more details using a CSS transition.
  • The site has a clean, responsive layout using Flexbox that wraps the movies nicely on different screen sizes.
