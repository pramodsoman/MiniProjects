Expansion Card

This is a project about "Expansion Cards" - an interactive expanding cards interface. When clicking on cards, there is a section showcasing the interactive expanding cards effect. When clicking on a card, it smoothly expands to reveal more content and information. The other cards contract accordingly with a smooth animation. This effect is achieved using HTML, CSS and JavaScript. The HTML provides the structure with different "div" elements for each card. The CSS handles the visual styling and animations. And JavaScript is used to add interactivity, toggling classes on click to expand and contract the cards.

Split-Landing-Page

This project implements a split landing page with a smooth drag-and-drop animation using JavaScript. When the page loads, two separate landing sections are shown side-by-side. Users can click and drag the divider line in between to reveal more of one section and less of the other. HTML and CSS are used to create the initial two-column layout with a divider element between. JavaScript listens for mousedown and mousemove events on this divider to calculate how much it should shift the two sections left and right on drag. Transition effects create a smooth animation between states.
