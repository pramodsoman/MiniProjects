Project-3

Form-Input-Wave

A simple login form with animated waving labels. When an input is clicked, the corresponding label's letters slide up and down in a smooth waving motion. This is achieved by using JavaScript to wrap each letter in a span and apply a transition delay to each span, offsetting the animation timing to create the wave effect. Specifically, the JavaScript splits the label's innerText into an array, maps over each letter to wrap it in a span with a transition delay based on the index, then joins them back together. CSS transitions on the spans animate the smooth up and down motion. The subtle waving effect adds visual interest to a standard login form. Built with HTML, CSS transitions, and JavaScript.

Project-4

Movie Discovery App

This is a simple movie discovery web app that allows users to browse and search for movies to see some basic information and overviews.

Features:

  • It uses the TMDB (The Movie Database) API to fetch popular movies and display them on the homepage.
  • The getMovies() function fetches the data from the TMDB API URL and passes it to showMovies() to display each movie.
  • For each movie, it displays the title, poster image, vote average, and overview. The CSS stylizes each movie container.
  • There is a search bar that allows the user to search for movies by name. The search calls the TMDB search API and displays matching movies.
  • The getClassByRate() function assigns a color class to the vote average span based on the rating value. Green for 8+, orange for 5+, red for below 5.
  • When a movie poster is clicked, the overview text slides up to reveal more details using a CSS transition.
  • The site has a clean, responsive layout using Flexbox that wraps the movies nicely on different screen sizes.

    • In summary, this is a simple movie discovery app that allows browsing popular and searching movies using the TMDB API. Users can view basic information and toggle overview details. The code uses asynchronous JS to fetch and render API data. The app has a minimal, easy to use interface with some nice styling effects.
